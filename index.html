Co to walidacja strony html?

Proces walidacji strony internetowej to sprawdzanie zgodności kodu HTML
witryny z obowiązującymi standardami sieciowymi (ustalonymi przez W3C)
w celu zapewnienia poprawnego działania strony na każdym urządzeniu i przeglądarce.



Co to walidator strony html?

Walidator to program sprawdzający błędy składniowe w kodzie.
Walidatory można tworzyć dla dowolnego formatu i języka,
ale w tym kontekście mówimy o narzędziach sprawdzających kod HTML , CSS i XML.

Możliwości formatowania tekstu w HTML

  • Przejście do nowego wiersz w tekście: < br >
  • Zapobieganie przejściu do nowego wiersza w tekście: < nobr > < /nobr >
  • Pólpauza: – & ndash;
  • Indeks górny: < sup > < /sup >
  • Indeks dolny: < sub > < /sub >
  • Przekreślenie: < strike >< /strike >
  • Pomniejszenie: < small > < /small >
  • Powiększenie: < big > < /big >
  • Dalekopis: < tt > < /tt >

Wielkości nagłówków w HTML

  1. Największy nagłówek

    2. : < h1 > < /h1 >

  2. Pomiędzy dużymi a małymi

    3. : < h3 > < /h3 >
  3. Najmniejszy nagłówek
    4. : < h6 > < /h6 >

Możliwości funkcji font

  1. < font size="Wielkość czcionki">
  2. < font color="Kolor czcionki">
  3. < font face="Typ czcionki">

Sposoby zmienienia koloru w funkcji font

  1. Nazwą: red=czerwony, green=zielony, blue=niebieski
  2. RGB: rgb(255, 0, 0)-czerwony, rgb(0, 255, 0)-zielony itd.
  3. Hexadecymalny: #RRGGBB
